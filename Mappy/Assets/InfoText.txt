Ctrl + 1 - select Sections tab
Ctrl + 2 - select Features tab
Ctrl + 3 - select Starts tab
Ctrl + 4 - select Height tab
Ctrl + 5 - select Void tab
Ctrl + 6 - select Attributes tab

Main Map
Pan - hold space, left-click and drag to pan the map
Shift + 1 through Shift-0 - centre on player start position

Height editing (all modes)
Shift + mouse wheel up - increase 'Cursor size'
Shift + mouse wheel down - decrease 'Cursor size'
Ctrl + Shift + mouse wheel up - increase 'Interval'
Ctrl + Shift + mouse wheel down - decrease 'Interval'

Height editing (set mode)
Ctrl + left-click - set 'Selected height'
Ctrl + Shift + mouse wheel up - increase 'Selected height'
Ctrl + Shift + mouse wheel down - decrease 'Selected height'

Void editing
Shift + mouse wheel up - increase 'Cursor size'
Shift + mouse wheel down - decrease 'Cursor size'
